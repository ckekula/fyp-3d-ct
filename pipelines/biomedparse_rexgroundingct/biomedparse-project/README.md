# Biomedparse Project

## Overview
The Biomedparse project is designed to facilitate the parsing and grounding of biomedical text using advanced natural language processing techniques. This project implements a pipeline that includes data loading, preprocessing, model training, evaluation, and visualization.

## Installation Instructions
To set up the project, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd biomedparse-project
   ```

2. **Create a virtual environment**:
   ```bash
   python -m venv venv
   ```

3. **Activate the virtual environment**:
   - On Windows:
     ```bash
     venv\Scripts\activate
     ```
   - On macOS/Linux:
     ```bash
     source venv/bin/activate
     ```

4. **Install the required packages**:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the pipeline, open the Jupyter Notebook located at `pipelines/biomedparse_rexgroundingct/pipeline.ipynb`. The notebook contains all the necessary code to execute the entire process from data loading to model evaluation.
