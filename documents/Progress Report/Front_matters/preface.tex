
\chapter*{Preface}


This report presents the research and development carried out as part of our final year group project on the interpretable analysis of 3D chest CT images. The project explores a sparse dictionary learning framework combined with label-consistent dictionary refinement to learn compact volumetric atoms that represent both normal lung structures and disease-related radiological patterns. The main focus of this work is interpretability, where the learned atoms help explain how the system makes diagnostic decisions based on visible image features.

The report includes the theoretical background, the proposed frozen dictionary learning method, implementation details, and an evaluation framework for classification, grounded localization, and explainability. Experimental results on curated 3D CT datasets show that the proposed method achieves competitive performance while providing better interpretability compared to common deep learning approaches.
