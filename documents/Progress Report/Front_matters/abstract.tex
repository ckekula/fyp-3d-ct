
\chapter*{Abstract}
Lung disease diagnosis from 3D chest CT scans requires clinically interpretable methods to support radiological decision-making.
This study introduces a sparse dictionary learning framework combined with Label-Consistent K-SVD (LC-KSVD) to learn compact volumetric atoms representing both normal anatomy and disease-specific pathologies.
The key contribution of this work is an interpretable representation in which learned dictionary atoms are associated with clinically meaningful volumetric patterns, such as ground-glass opacity and consolidation, allowing diagnostic decisions to be traced back to relevant anatomical structures
The method is benchmarked against state-of-the-art deep learning models using a curated 3D chest CT dataset with expert annotations for multi-class abnormality classification and grounded localization.
A unified evaluation framework is developed to assess classification accuracy, grounded localization, and explainability across heterogeneous models using a standardized adapter-based pipeline. Experimental results demonstrate competitive classification performance while providing improved interpretability and clinically aligned localization compared with deep learning baselines.
