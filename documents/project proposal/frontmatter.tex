\chapter*{Abstract}
Pulmonary disease diagnosis using 3D chest CT faces key challenges related to computational efficiency and clinical interpretability.
Traditional 3D Convolutional Neural Networks require extensive labeled data and GPU resources, and post hoc explainability methods
such as Grad CAM do not provide sufficient granularity for medical decision making.
Although deep learning models achieve high accuracy, their black box nature and large computational demands limit clinical
adoption. This project proposes a sparse dictionary learning framework that addresses these limitations through inherently
interpretable feature representations. The methodology uses a frozen dictionary learning approach combined with
Label Consistent K-SVD to capture both normal anatomical patterns and disease specific pathological features. Dictionary
atoms are mapped to radiological features such as Ground Glass Opacity and consolidation, enabling transparent
diagnostic reasoning. Each dictionary atom will correspond to specific volumetric patterns, allowing clinicians to trace classification
decisions back to anatomical features rather than relying on post hoc visualization techniques.
The proposed system will be benchmarked against standard K-SVD and 3D CNN baselines using the CT-RATE and ReXGroundingCT datasets.
The sparse representation framework naturally handles class imbalance and requires fewer training samples than deep learning
approaches, making it particularly suited for rare pathologies. This research demonstrates that sparse representation methods can
provide clinically interpretable lung disease classification while maintaining competitive performance with reduced computational complexity.
